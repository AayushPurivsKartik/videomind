# VideoMind - AI That Watches Videos For You

**Upload any video. Ask anything in English. Get exact moments.**

Examples:
- "Show me when someone enters the room"
- "Find every frame with more than 3 people"
- "When did the dog jump?"
- "Count how many times he waved"

## Features
- Florence-2 for rich frame descriptions
- YOLOv8 real-time object detection
- CLIP + FAISS semantic search
- Beautiful Gradio interface
- 100% open source & local

## Run Locally
```bash
git clone https://github.com/AayushPurivsKartik/videomind.git

pip install -r requirements.txt
